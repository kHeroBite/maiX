- 캘린더/채팅도 동일 증상

### 문제 2 원인

`MainWindow.xaml.cs:16639` `ShowSyncMailSettings()`:
- 즐겨찾기 라디오: `IsChecked = mailIntervalSeconds == seconds`
  - `mailIntervalSeconds`는 `prefs.MailSyncIntervalSeconds`(기본값 0) 기반 → 항상 미선택
  - 올바른 비교: `prefs.FavoriteSyncIntervalSeconds == seconds` (기본값 10)
- 전체 라디오: 같은 문제, `prefs.FullSyncIntervalSeconds == seconds` (기본값 300)로 비교해야 함
- Checked 콜백: 두 그룹 모두 `SetSyncInterval` 호출 → `SetFavoriteSyncInterval` / `SetFullSyncInterval` 분리 필요
- prefs 저장도 `MailSyncIntervalSeconds` 대신 `FavoriteSyncIntervalSeconds` / `FullSyncIntervalSeconds` 사용해야 함
